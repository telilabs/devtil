# Devtil

A collection of small developer utilities that run entirely in the browser.

**[→ Open the app](https://telilabs.github.io/devtil/)**

## Tools

- **JSON Formatter** — pretty-print, minify, validate, sort keys
- **cURL → OpenAPI** — convert curl commands to an OpenAPI 3.0.3 spec
- **OpenAPI Cleaner** — strip unused paths and components from a spec
- **Postman Minimizer** — trim a Postman collection down to selected requests
- **JSON ↔ YAML** — convert between formats with auto-detection
- **Base64** — encode/decode text and binary files
- **JWT Decoder** — inspect header, payload, and claims (no signature verification)
- **Timestamp** — convert between Unix timestamps, ISO 8601, and human-readable formats
- **Generators** — UUID v4 and MD5/SHA hashes
